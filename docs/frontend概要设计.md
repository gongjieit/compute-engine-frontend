主要功能包括：

### 核心功能模块
1. **简单计算** - 加减乘除运算
2. **复杂算法** - 平均值、方差、相关系数、斐波那契、阶乘等
3. **金融计算** - 最大回撤、时间加权收益率、波动率计算
4. **异步任务** - 异步计算任务创建
5. **任务管理** - 任务状态查询、记录列表、统计分析、任务面板

### 特色设计亮点

1. **响应式设计** - 支持桌面端和移动端访问
2. **实时反馈** - 计算结果即时显示
3. **任务追踪** - 异步任务状态实时监控
4. **数据可视化** - 使用图表展示计算结果和统计信息
5. **批量操作** - 支持多组数据的批量计算
